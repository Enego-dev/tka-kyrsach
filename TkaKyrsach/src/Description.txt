2. PROCESS_CONTROL_CODE
    Автомат условно говоря работает по циклу операция1 -> операция2 -> операция3 -> операция4

    Код операции представляет собой 3-битовое двоичное число, например 101. Если позиция четная, то это загрузка, в противном случае - выгрузка

    // 0b101, 0b101, 0b011, 0b010
    Теперь конкретно по реализации:
        1. Увидел код
        2. Определяю что за операция (load / unload), копирую маску управляющего кода
        3. Перемещаю контейнер в DPi.
        4. Если загрузка:
            4.1 Можно выдавить материал из резервуара?:
                4.1.1 Можно: Открываю клапан, выливаю материал по таймеру, закрываю клапан, помечаю бит как "0" - выполнено
                4.1.2 Нельзя: перехожу к следующему биту
            4.2 Проверка на ожидаемое исполнение операции (если все отработано четко):
                4.2.1 Перехожу к следующему управляющему управляющему коду
                4.2.2 Еду на слив отходов (т.н. код "000"), перехожу к следующему управляющему коду загрузки
            4.3 Устанавливаю текущий управляющий код на -1, как отработанный
        5. Если выгрузка:
            5.1 Можно перелить в бункер?
                5.1.1 Можно: переворачиваю контейнер, выливаю, переворачиваю контейнер
                5.1.2 Нельзя: Еду на слив отходов (т.н. код "000"), к следующему управляющему коду
            5.2 Устанавливаю текущий управляющий код на -1, как отработанный
